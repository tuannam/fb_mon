📬 Telegram Setup Guide

1

Create a Telegram Bot

Open Telegram and search for @BotFather.

Send the command /newbot, then follow the prompts to choose a name and username for your bot.

BotFather will reply with your Bot Token — it looks like:
123456789:ABCdefGhIJKlmNoPQRsTUVwxYZ

Copy that token and paste it into the Telegram Bot Token field.

2

Get your Chat ID

Important: Open Telegram, search for your bot by its username, and send it a message — e.g. hello. You must do this before clicking the button below, otherwise the result will be empty.

⚠️ If you already clicked and got {"ok":true,"result":[]} — go send a message to your bot in Telegram first, then click again.

Then click the button below to look up your Chat ID:

In the page that opens, find "chat":{"id": — that number is your Chat ID.

Example: {"id": 987654321, "first_name": "Nam"}

⚠️ The URL must start with https://api.telegram.org/botYOUR_TOKEN/getUpdates — note the bot prefix.

3

Paste & Test

Paste both values into the fields, click Save settings, then click Test Telegram to confirm it works.
